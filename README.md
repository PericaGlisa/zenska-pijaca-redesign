# Ženska Pijaca Redesign

Projekat za redizajn web prodavnice Ženska Pijaca.

## Tehnologije

Ovaj projekat je izgrađen koristeći:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Lokalni razvoj

Pratite ove korake za pokretanje projekta lokalno:

```sh
# 1. Instalirajte zavisnosti
npm install

# 2. Pokrenite razvojni server
npm run dev
```
